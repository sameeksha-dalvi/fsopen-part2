# Full Stack Open – Part 2

This repository contains my solutions for Part 2 of the Full Stack Open course.  
Part 2 focuses on **communicating with a server** using React, including fetching and sending data, handling forms, managing state, and implementing notifications.

## Exercises Completed

**2.1-2.5: Rendering Courses & Calculations**  
Rendered courses and calculated total exercises. Refactored components into modules.

**2.6–2.10: Phonebook App**  
Created a phonebook app with add, update, delete, and search functionality. Split UI into reusable components.

**2.11–2.15: Server Communication**  
Fetched and saved phonebook entries from/to backend using REST API. Handled updates, deletions, and errors.

**2.16–2.20: Countries App & Phonebook App Notifications**  
Fetched country data from an API, displayed details, added weather info, and for phonebook app implemented notifications for user actions.
